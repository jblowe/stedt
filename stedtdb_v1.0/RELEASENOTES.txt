v1.0 11 June 2017

1. The primary deliverable of the STEDT project, the published version
of the Dictionary-Thesaurus, has been published.  A very limited run of 15 "hardcopies", in a handsome imitation leather binding, is currently being prepared at UC Berkeley, and will be available for purchase for approximately 400 USD in the coming weeks.

Let us know if you’d like to order a copy!

The  “camera-ready” PDF 1,185 pages (xlii, 1141 pp.) in 11”x17” format, is available for download. The file is 22MB.

The PDF is contained in the archive mentioned in 2. below

And also available online at:

   https://berkeley.box.com/s/fnox5ywkhg4haq78hrvzilbf207rfju7

2. A "side effect" of the project is the substantial database of lexicons of Sino-Tibetan languages, the "STEDT database". In its current incarnation, it is a MySQL database with a (mostly) normalized schema.

Note, that the database is already available to be searched online, and has been for some time:

   http://stedt.berkeley.edu/search

For example, here is a example of a cognate set PTB *d-kʷəy-n 'DOG'

   https://stedt.berkeley.edu/~stedt-cgi/rootcanal.pl/etymon/1764

However, the database and software to manipulate this database have not been widely distributed until now. They are now being made publicly available under a generous license.

NB: the software is already available in GitHub:

    https://github.com/stedt-project/sss

E.g. the PERL code implementing the “primary UI” mentioned in the links above:

    https://github.com/stedt-project/sss/tree/master/rootcanal

Note that deploying this code is not for the faint of heart: little engineering has been done to make sure the code works in different environments and can be installed reliably. I am happy to help anyone who would like to make their own deployment.

The database itself is available for download as a compressed database dump, along with minimal documentation and some "derivative versions". It is in the same Box folder as the PDF mentioned in 1.

It should be in the following folder:
