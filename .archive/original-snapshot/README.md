# Golden corpus: frozen pages from the original STEDT web app

Raw HTML scraped from the last live deployment of the original Perl/MySQL STEDT
application ("rootcanal", https://stedtdb.johnblowe.com/) on **2026-06-10**, before it
disappears. This is the reference for what the original actually rendered — the oracle
for parity work on the remake. Files are byte-faithful response bodies; nothing has been
reformatted.

- `MANIFEST.tsv` — every archived URL: category, local file, and why it was selected.
- `FETCHLOG.tsv` — per-fetch status, byte count, sha256, and elapsed time (all 460 were HTTP 200).
- `scrape.py` — the scraper that produced this (sequential, 1.2 s throttle; rerunning
  skips existing files, so it can top up the corpus while the original is still alive).
- `pages/` — the corpus:
  - `etymon/{tag}.html` — 300 etyma chosen for size extremes, exemplary/provisional
    status, note edge cases (`<unicode>`, quotes, brackets, italics, cf.-xrefs),
    allofams, mesoroots, HPTB links, protoform alternations, chapter spread, random fill.
  - `language/{lgid}.html` — 50 languages. The original has no language page; its
    language view is the gnis search filtered by `lexicon.lgid`, so these are
    `/gnis?lexicon.lgid={lgid}` results.
  - `source/{abbr}.html` + `source-index.html` — 30 source pages and the bibliography index.
  - `group/{n}.html`, `group/{a}.{b}.html` — every top-level group page + 15 subgroups.
  - `chap/{semkey}.html` + `chapters.html` — 28 chapter pages across nesting depths
    (near-empty ~2.4 kB pages are real: chapters with no etyma render only their title).
  - `search/*.html` — a 20-query gnis battery (gloss, form, language, lggrp, source,
    combos, comma-OR, diacritics, empty result).
  - `home.html`, `gnis-form.html` — splash page and the full search form.
- `assets/` — the site's CSS plus `stedtconfig.js` (client-side syllabification etc.),
  kept so the markup in `pages/` stays interpretable.

Selection was driven from this repo's own database (same underlying dump), seeded RNG
(20260610), generated by the manifest block recorded in the repo history alongside this
commit. Logged-out view throughout (no session), so edit-only UI is absent by design.
